Q29.[Crypto] Common World

解いた方法
c1 = m^e1
c2 = m^e2
ここで　c2%c1==0　であったので
m^(e2-e1) = c2//c1
具体的には
m^(13-11) = 180040032052240663195289808096837316

m^2を小さい数で割ると
2*2,11*11,197*197,1597*1597
の倍数であった
その後sqrtを取ってみると
61304268923*61304268923で割れることも分かった
よって
m = 2*11*197*1597*61304268923 = 424311244315114354


別解
m^2のルートを二分探索で求める！
m2 = c2//c1
print(c2%c1)
print(m2)
ok = 1;
ng = m2;
while ng-ok>1:
  mid = (ok+ng)//2
  if mid*mid>m2:
    ng = mid
  else:
    ok = mid
print(ok)
